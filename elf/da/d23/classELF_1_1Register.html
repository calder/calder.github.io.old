Detailed Description

Storage for functions.

Remarks:
  • A "bucket" is an ordered list of "slots", each containing a Pointer
  • to some Value (or another pointer).
  • A "register" stores a single "bin" for each type".
The Pointer objects in a bucket are fixed after initialization, but the Values they point to are free to change.
