Detailed Description

Parent to all Instructions available in a Language.

Remarks:
Instructions intentionally don't store their arguments in reference counting Ptr's, because they should only be referencing Pointers that belong to Registers. While the resulting seg-faults are not pretty, the alternative behavior would be even more confusing.

virtual int ELF::Instruction::getStage (  )  const [inline, virtual]

Return the "stage" (default 0) in which this instruction is executed.

Note:
When a Function is called, it makes increasing passes through each stage, calling all Instructions within that stage.
For almost all Instruction types, this should be 0. The best example of something that needs an earlier stage, though, is the LabelInstruction.
