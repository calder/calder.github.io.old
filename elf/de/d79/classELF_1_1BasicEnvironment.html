Detailed Description

A standard evolutionary computing environment.

Note:
Specifications:
  • Mutation: Parents are copied to generate a child population. Each child is mutated as determined by the MutationScheme.
  • Selection: Parents and children are considered together in a lambda + mu scheme. The specific selection method is configurable with setSelectionMethod(), the default being tournament selection.
    • Tournament: n-way tournament, n set by setTournamentSize(). The default is 4.
    • Roulette: Fitness weighted roulette. Weight function is set by setRouletteWeightFunction(). The default is linear.
