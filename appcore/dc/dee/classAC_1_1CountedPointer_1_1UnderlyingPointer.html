Detailed Description

template<class T>
class AC::CountedPointer< T >::UnderlyingPointer

Implementation class for CountedPointer.

Note:
When CountedPointer objects are created, they create an UnderlyingPointer object that they store a reference to. Thus when you copy a CountedPointer object, it simply stores another reference to the UnderlyingPointer object. When the last CountedPointer object for a given UnderlyingPointer is destroyed, the UnderlyingPointer will call the destructor for its payload.
