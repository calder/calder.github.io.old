Detailed Description

Parent class for all classes wishing to make use of SerialData system.

Remarks:
Serializable objects store all their member variables and children in SerialData objects, allowing their complete states to be easily written to and read from a string or file. The Serializable class also provides generic loading from SerialData objects by allowing users to register their own classes for automatic initializaiton whenever a particular type is encountered in loading a Serializable tree from a SerialData object. To achieve this, all user classes registering themselves must define the serial constructor outlined below.
