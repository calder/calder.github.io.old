virtual void AC::MessageHandler::processMessage ( const Message &  message) [inline, virtual]

Called directly by a MessageCenter to alert object.

Note:
The default implementation simply calls handleMessage() with the given message. However, by overriding this method, you may create custom classes of MessageHandlers with their own behavior. For example, the ThreadBoundMessageHandler overrides this method to handle the message in its parent thread rather than handling it immediately.
