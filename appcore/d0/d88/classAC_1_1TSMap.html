Detailed Description

template<class K, class V>
class AC::TSMap< K, V >

Thread-safe wrapper for std::map.

Remarks:
To avoid potential confusion, TSMap iterators procure read-write locks on creation which are only released on destruction. This is for you own good. It also means that only one thread can maintain iterators to a given object at any moment, so be sure you understand the weird deadlock opportunities this introduces.
