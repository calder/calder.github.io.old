Detailed Description

template<class RETURN>
class AC::ReturnCallable< RETURN >

A Callable that can be queried for the return value of the function.

Note:
WARNING: Objects of this type are not automatically deleted by the thread they are queued in and must be destroyed by the queueing thread.
